# taulukko-commons-cql
Common CQL Interpreter


Import and deploy
====================

Add in your maven ( settings.xml ) the reposotory below:

  <mirrors>	
	<mirror>
			<id>Central Internal</id>
			<url>http://repository.taulukko.com.br:8080/repository/internal</url>
			<mirrorOf>*</mirrorOf>
	</mirror>	
  </mirrors>
